微信小程序日历内容显示功能

在日期下方显示数据内容，支持自定义背景色、文字颜色

功能状态

  • 支持在日期下方显示数据内容
  • 支持自定义背景色和文字颜色
  • 支持单个和多个内容项
  • 支持字符串和对象两种内容格式
  • 支持内容行数限制
  • 支持动态添加/删除/更新内容
  • 完整的 TypeScript 类型支持
  • 向后兼容现有功能
内容显示已启用

代码示例

// 基础用法
WxCalendar.use(DateRangePlugin, {
  ranges: [
    {
      name: '项目开发',
      code: 'project_dev',
      startDate: '2024-03-01',
      endDate: '2024-03-15',
      content: '前端开发阶段\n进度：65%'
    }
  ],
  showContent: true,
  contentMarkAs: 'schedule'
});

// 高级用法 - 多内容项
{
  name: '产品发布',
  code: 'product_release',
  startDate: '2024-04-01',
  endDate: '2024-04-05',
  contents: [
    { text: '✅ 代码审查完成', color: '#00b894' },
    { text: '🔄 部署准备中', color: '#fdcb6e' },
    { text: '📋 文档待更新', color: '#6c5ce7' }
  ]
}
